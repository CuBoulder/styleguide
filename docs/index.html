The CU Boulder Web Style Guide is a comprehensive resource that provides descriptions and code for:

  • how to apply the CU Boulder Web Brand Identity Standards;
  • and, a collection of reusable styles and components for use on official campus web projects.

The design and content of this style guide is a work in progress, but is being actively worked on.

View this project on Github Download

Brand Guidelines

The web brand identity standards must be used on all official university websites that are using a colorado.edu domain. The CU Boulder web brand identity standards comprise of four key areas:

  • Header elements
  • Footer elements
  • Web font
  • Color pallette

Web Style Components

A comprehensive style guide for not only implementing the CU Boulder brand but provide a unified experience through common design themes and approved elements of color, typography, iconography, forms and layouts.
