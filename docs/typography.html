Embedding Web Fonts

The official brand font is Helvetica Neue. There is a cost associated with using Helvetica Neue as a web font. Subdomain websites can opt to use the secondary approved brand web font, Roboto, which is free (website usage only).

Subdomain sites are encourage, but not required, to implement Roboto for their web font. Roboto is a free font maintained by Google and you can add the font to your site with the code below:

HTML

  
    <link href="https://fonts.googleapis.com/css?family=Roboto:400,700|Roboto+Condensed:700" rel="stylesheet">

The CU Boulder styleguide CSS already includes the proper references to this webfont, but you can also define the fonts yourself with the following css:

CSS

  
    font-family: 'Roboto', sans-serif;

    font-family: 'Roboto Condensed', sans-serif;

.h1 through .h6 classes are also available, for when you want to match the font styling of a heading but cannot use the associated HTML element.

HTML

  
    <p class="h1">Heading Level 1</h1>
    <p class="h2">Heading Level 2</p>
    <p class="h3">Heading Level 3</p>
    <p class="h4">Heading Level 4</p>
    <p class="h5">Heading Level 5</p>
    <p class="h6">Heading Level 6</p>

Paragraphs

This is a normal paragraph. On static pages, it's designed for readability and optimizing the number of characters per line. The font size is larger to account for longer chunks of text.

This is a lead paragraph. It might be used as a longer-form tagline or subheading for a page or content section. It has a slightly larger font size and line-height.

HTML

  
    <p class="lead">This is a lead paragraph.</p>

This is a hero paragraph. It might be used to get a user's attention.

HTML

  
    <p class="hero">This is a hero paragraph.</p>

This is a supersize paragraph. It might be used in infographics to drastically increase the font size.

HTML

  
    <p class="supersize">This is a supersize paragraph.</p>
