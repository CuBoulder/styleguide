Currently, Web Express sites on www.colorado.edu have access to the brand fonts. For sites on subdomains or other urls we suggest using the the Roboto web font.

You can embed our recommended web fonts with the code below:

<link href="https://fonts.googleapis.com/css?family=Roboto|Roboto+Condensed:700" rel="stylesheet">

The CU Boulder styleguide CSS already includes the proper references to this webfont, but you can also define the fonts yourself with the following css:

font-family: 'Roboto', sans-serif;
font-family: 'Roboto Condensed', sans-serif;
