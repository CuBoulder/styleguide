Stylesheets

There are a few stylesheet options depending on the needs of your project:

All the Things: This is the base theme file for the CU Boulder Web Express platform. This file includes many classes that are unnecessary outside of the platform and some reduntant styles to make using this styleguide easier. While you can implement the CU Boulder styles with other frameworks, you may encounter some issues with conflicting styles.

Helper Classes

Below are some utility classes to help with adding various style attributes to any element:

Margins

  • .margin-top - adds a 20px top margin
  • .margin-bottom - adds a 20px bottom margin
  • .margin-left - adds a 20px left margin
  • .margin-right - adds a 20px right margin

Padding

  • .padding - adds 20px padding
  • .padding-small - adds 10px padding
  • .padding-large - adds 40px padding
  • .padding-none - removes all padding from element

Directional Padding

  • .padding-top, .padding-top-small, .padding-top-large
  • .padding-bottom, .padding-bottom-small, .padding-bottom-large
  • .padding-left, .padding-left-small, .padding-left-large
  • .padding-right, .padding-right-small, .padding-right-large
  • .padding-vertical, .padding-vertical-small, .padding-vertical-large
  • .padding-horizontal, .padding-horizontal-small, .padding-horizontal-large

Borders

  • .border - adds a border to all sides
  • .border-top - adds a top border
  • .border-bottom - adds a bottom border
  • .border-left - adds a left border
  • .border-right - adds a right border

Text Size

These styles will only work with text that is not set in explicit pixel values.

  • .text-magnify - increases font size by 15%
  • .text-minify - reduces font size by 15%

Alignment

These classes will float an element to the left or right. They should be used in combination with a margin class to add spacing around the element.

  • .align-left - floats the element to the left.
  • .align-right - floats the element to the right.

Clearing aligned/floated elements

In order to properly clear a floated element, the .clearfloats class should be added to the parent element.
