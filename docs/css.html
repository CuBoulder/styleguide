Stylesheets

There are a few stylesheet options depending on the needs of your project:

All the Things: This is the base theme file for the CU Boulder Web Express platform. This file includes many classes that are unnecessary outside of the platform and some reduntant styles to make using this styleguide easier. This file does not include any grid classes to minimize conflicts with other frameworks. If you need grid classes, use the grid css listed below.

While you can implement the CU Boulder styles with other frameworks, you may encounter some issues with conflicting styles.

There are also some smaller collections of the styleguide that can be applied as needed:

  • Branding
  • Buttons
  • Colors
  • Grid & Layout
  • Typography

We recommend downloading the stylesheets and adding them to your project rather than try to reference them from this website directly.

Utility Classes

There are a variety of utility or helper classes that can provide alignment, margins, padding, text sizing, and visibility which can be applied to any html.
