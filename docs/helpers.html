The styleguide contains a variety of utility or helper classes that can be used rather than having to make custom styles of your own.

Padding

The padding classes can be added to any element to add padding. There are three sizes available (small/10px, medium/20px, large/40px) and a direction can be specified.

All Directions

  • .padding-small
  • .padding/.padding-medium
  • .padding-medium

Direction Specific

You can combine the size (small, medium, large) with a direction (top, bottom, left, right) to add padding to a specific part of an element. For example:

  • .padding-small-top
  • .padding-medium-left
  • .padding-large-bottom

You can also specify the padding to be either vertical or horizontal:

  • .padding-horizontal-small
  • .padding-vertical-large

Removing Padding

There are also three utility classes for removing padding from an element:

  • .padding-none
  • .padding-none-vertical
  • .padding-none-horizontal

Margins

The margin classes can be used to create space between one element and another.

  • .margin-top, .margin-top-large
  • .margin-bottom, .margin-bottom-large
  • .margin-left, .margin-left-large
  • .margin-right, .margin-right-large

Removing Margins

Sometimes you may need to remove a margin from an element.

  • .margin-none
  • .margin-none-horizontal
  • .margin-none-vertical
  • .margin-none-bottom

Borders

The border classes can be used to add a border on to all or specific sides of an element:

  • .border/li>
  • .border-top
  • .border-bottom
  • .border-left
  • .border-right

Borders can also be removed from all or specific sides of an element:

  • .border-none
  • .border-top-none
  • .border-bottom-none
  • .border-left-none
  • .border-right-none

Alignment

An element can be aligned to the left or right uses the following classes:

  • .align-left
  • .align-right

Text in an element can be aligned with the following classes:

  • .text-align-left
  • .text-align-right
  • .text-align-center

Sizing

The text size of an element can be sized up or down with the folling classes. These are meant to be small adjustments to make something slightly larger or somaller.

  • .text-magnify
  • .text-minify
