Adding Icons

If your project requires the use of icons, we suggest using the Font Awesome icon library. You can add the icon font to your website with the code below:

HTML

  
    <link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.3.1/css/all.css" integrity="sha384-mzrmE5qonljUremFsqc01SB46JvROS7bZs3IO2EmfFsd15uHvIt+Y8vEf7N7fWAU" crossorigin="anonymous">

How to use Font Awesome

Icon Colors

There is a separate set of colors for icon use.

{% for icon-colors in site.data.icon-colors %} {% endfor %}

Usage

To use the icon colors, just apply the icon color class to the icon. The icon will be the same color as the text for the element if no color is defined. For example, if the background of the element is back, the text color would be white and the default icon color would be white.

{% for icon-colors in site.data.icon-colors %}

{{ icon-colors.color }}

  
    <i class="fas fa-stroopwafel fa-3x {{ icon-colors.class }}"></i>
{% endfor %}
