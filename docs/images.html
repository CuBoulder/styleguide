The classes below include a variety of ways to size images in pixel and percent-based widths.

Image Pixel Widths

{% assign px = "50|75|100|150|200" | split: '|' %} {% for i in px %}

{{ i }}px

.image-{{ i }}px

HTML

  
    <img class="image-{{ i }}px" src="http://via.placeholder.com/1600x900" alt="placeholder image" />
{% endfor %}

Responsive Image Pixel Widths

Extra Small (xs)

    {% for i in px %}
  • .image-xs-{{ i }}px
    • {% endfor %}

Small (sm)

    {% for i in px %}
  • .image-sm-{{ i }}px
    • {% endfor %}

Medium (md)

    {% for i in px %}
  • .image-md-{{ i }}px
    • {% endfor %}

large (lg)

    {% for i in px %}
  • .image-lg-{{ i }}px
    • {% endfor %}

Example

<img class="image-lg-200px image-md-150px image-sm-100px image-xs-50px" src="http://via.placeholder.com/1600x900" alt="placeholder image" />

Image Percentage Widths

Using these classes will apply a consistant image width to all screen sizes. Images will always scale relative to the size of the container element.

{% assign sizes = "25|33|50|66|75|100" | split: '|' %} {% for i in sizes %}

{{ i }}%

.image-{{ i }}

HTML

  
    <img class="image-{{ i }}" src="http://via.placeholder.com/1600x900" alt="placeholder image" />
{% endfor %}

Responsive Image Percentage Widths

Using these classes you can assign an image a different width for each breakpoint.

Extra Small (xs)

    {% for i in sizes %}
  • .image-xs-{{ i }}
    • {% endfor %}

Small (sm)

    {% for i in sizes %}
  • .image-sm-{{ i }}
    • {% endfor %}

Medium (md)

    {% for i in sizes %}
  • .image-md-{{ i }}
    • {% endfor %}

large (lg)

    {% for i in sizes %}
  • .image-lg-{{ i }}
    • {% endfor %}

Example

HTML

  
    <img class="image-lg-25 image-md-33 image-sm-50 image-xs-100" src="http://via.placeholder.com/1600x900" alt="placeholder image" />

Image Alignment

Using these classes you can align or float an image in different directions for each breakpoint.

Extra Small (xs)

  • .image-xs-align-left
  • .image-xs-align-right
  • .image-xs-align-none

Small (sm)

  • .image-sm-align-left
  • .image-sm-align-right
  • .image-sm-align-none

Medium (md)

  • .image-md-align-left
  • .image-md-align-right
  • .image-md-align-none

large (lg)

  • .image-lg-align-left
  • .image-lg-align-right
  • .image-lg-align-none
